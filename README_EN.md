# AEVE-Plugins

## Overview

This is a modernization and acceleration implementation of the Adobe
AfterEffects plugin
"[F's Plugins](https://github.com/bryful/F-s-PluginsProjects)" for animation
production.\
🚧 Work in progress 🚧
